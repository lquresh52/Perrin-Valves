KH-11

KH-11

Nuclear power applications demands a high corrosive resistant materials with robust design for long lasting service life.

Perrin KH 11- S ball valve design features a three piece body and a floating, seat supported ball. The stem packing is spring loaded and the seat rings with pressure relief warranty are pre-loaded. Stem extensions, locking devices and actuators with accessories, can be attached without operating interruptions.

Salient Features
Size Range 1/2 - 4 inch / DN 15 – 100
Pressure Class Class 150 – 300
Temperature Range -60°C to 200°C
Testing Standards API 598, EN 12266 – 1/2
Certificates ‘TA – Luft’ Low Fugitive Emission
Actuator Mounting as per ISO 5211
Design Standards Fire safe API 607, API 6D, BS 6755 and ISO 10497
Stem Blow out proof stem, Live loaded stem packing
